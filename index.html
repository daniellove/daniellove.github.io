About me

I’m a Web Developer & Designer working in education. I specialise in Front-end Design and future proofing content for students from all walks of life. I am an expert in using CSS3 and Javascript with jQuery which makes creating interactive and responsive websites second nature. I also have experience in back-end web development using PHP, Ajax, API and multiple CMS such as Silverstripe, Squiz and Brightspace. Check out my full CV here.


Efficiency is one of my passions so naturally automation is a hobby of mine. From this I have automated and optimised Human Resource and project management processes using VBA and Microsoft Excel and created mobile application ‘bots’ using other ‘basic’ programming languages and image recognitions.
