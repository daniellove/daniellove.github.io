About me

I’m a Web Designer & Developer working in education. I specialise in UX design and future proofing content for students from all walks of life. I am an expert in using CSS3 and Javascript with jQuery which makes creating interactive and responsive websites are a breeze. I also have some experience in back-end web development using PHP, Ajax and multiple CMS such as Silverstripe, Squiz and Brightspace.


I have a passion for efficiency so automation applications are a one of my hobbies. From this interest I have optimised Human Resource and project management processes and created mobile application ‘bots’ using ‘basic’ programming languages such as VBA and AHK.
