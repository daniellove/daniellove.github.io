Purpose

This page contains a selection of tools to be used throughout NCEA modules. Please select any you would like to use or adapt for use and include them in your module request.

Media

IMAGES & captions

Images can be displayed in various different sizes and captions can be added where requested. Note that there are three different types or caption. The stand caption requires no specified attributes; however, there is a "dark" and a "light" caption if preferred for specific images.

It is important when considering images you will be using in your content. Always try to find something that is relevant to the children and what they need.

VIDEO
