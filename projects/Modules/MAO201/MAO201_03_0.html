New vocabulary

Here are some more nouns that will help you when talking about the weather.

hukapapa frost
kapua cloud
kōuaua shower (of rain)
kohu fog

We add ‘he’ in front to mean ‘a, an, or some’.

Example:
He hukapapa.
A/some frost.

IT’S ALL ABOUT THE WEATHER

You have now learned three different sentence structures to use to talk about the weather. You can:

  • describe the weather in the present tense
  • say what the weather is doing in the present tense
  • talk about what kind of day it is.

You also learned a few nouns to help when talking about the weather.

Click on the images to review these sentence structures.




Kei te + tūāhua.

It’s + adjective.

For example:

Kei te wera.

It’s hot.




Kei te + tūmahi + te + tūingoa.

The + noun + is + verb.

For example:

Kei te whiti te rā.

The Sun is shining.




He rangi + tūāhua.

A/an + adjective + day.

For example:

He rangi paki.

A fine day.
